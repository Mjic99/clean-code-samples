# Description

This repo shows a Log system implementation and a tic tac toe game implementation.  
These examples apply some of the principles exposed in the Clean Code book.

# Logger Typescript project

## Requirements
- Node.js v14.16.1 or later
- NPM v8.13.2

## Setup
1. Go to the folder
```
cd logger
```
2. Install requirements
```
npm install
```
3. Run the project
```
npm run start:dev
```
4. Make a GET request to localhost:3000/ and look at the console to see the execution results

# Tic Tac Toe PHP project

## Requirements
- PHP v7.4 or later 

## Setup
1. Go to the folder
```
cd tic-tac-toe
```
2. Run the script
```
php main.php
```
3. The console will ask for input, type it in the console in "x,y" format
